## 三条线路路由逻辑，一次说清楚

### LAX Premium（CN2 GIA）

**去程**：电信和联通走 CN2 GIA（AS4809），移动走 CMI 绕香港  
**回程**：三网全部走 CN2 GIA

这是 DMIT 洛杉矶的旗舰线路。CN2 GIA 是电信运营的精品跨国专线，59.43 开头的 IP 段是它的标志。路由追踪里看到 `59.43.*.*` 就说明确实走了这条路。

晚高峰实测下来，电信方向平均延迟在 130ms 左右，全国平均大概 155-165ms，丢包基本清零。这个数字在美西机房里属于很顶的水平——毕竟物理光速极限大约是 120ms，能做到 130ms 是真的贴线跑。

IP 全系是美国原生，精确识别为 Los Angeles, CA。

### LAX Eyeball（CMIN2 + 9929）

**去程**：电信联通走 CN2/AS9929，移动走 CMI  
**回程**：电信/联通走 AS9929，移动走 CMIN2

这条线路是 DMIT 在 Premium 之下加出来的中间档，定位就是给觉得 CN2 GIA 太贵但又不想将就普通线路的用户。

联通和移动用户用 Eyeball 体验会更突出——9929 是联通的精品政企线路，CMIN2 是移动的对标 CN2 GIA 的新专线，两个方向都不走普通骨干。晚高峰从北京电信 ping 测试 IP `154.17.226.2`，平均延迟在 140-180ms 之间，丢包率通常低于 5%。

比 Premium 便宜，线路质量比普通 163 好一档。日常跑博客、搭节点，够用。

### LAX Tier 1（国际 BGP）

**去程/回程**：电信走 CT163，联通走 CU169（AS4837），移动走 CMI

这条没有任何专门的中国大陆优化，就是纯国际线路直连。对国内用户来说，延迟会高一些，晚高峰也更容易受影响。

但 Tier 1 有一个优点：流量超额不直接关机，降速到一定带宽继续跑。如果你做的是国际业务或者大流量场景，这条线路性价比会更高。新平台跑的是 AMD EPYC 9005 系列，带宽最高 10Gbps。

## 测了数据之后，怎么选

说实话，测试 IP 的意义就是帮你做这个判断。

**电信用户**：Premium 是最直接的选择。CN2 GIA 双向优化，电信去回程都走专线，晚高峰延迟基本稳住不飘。Eyeball 对电信的优化程度稍弱一些，但省钱。

**联通 / 移动用户**：Eyeball 其实更值得考虑。9929 对联通来说是政企线路，比 CN2 GIA 并不差多少；CMIN2 是移动自己的精品线路，效果类似 CN2 GIA 但成本低一档。价格比 Premium 便宜，适合这两类运营商。

**不在乎国内延迟，主要跑国际业务**：Tier 1。便宜、带宽大、流量多，超额不断线。大流量 V4C4G 那个套餐 $36.90/月给 20TB，做 CDN 源站或者国际站再合适不过。

顺便说一点：DMIT 明确不超卖，热门套餐缺货是常事。看到合适的配置有库存，别犹豫太久。

## 几个常见问题

**Q：测试 IP 和我买到的 IP 会是同一个段吗？**

不一定。DMIT 的 IP 主要来自 154.17.x.x 段和 154.21.x.x 段，分配有一定随机性。但测试 IP 和实际 IP 走同一套线路基础设施，延迟结果有参考价值。

**Q：IP 被墙了怎么办？**

每 15 天可以免费申请换一次。非被墙原因换 IP 收 $5。这个政策比很多商家要好——不是完全自助，要走工单申请，但换回来的速度还可以。

**Q：买之前还有什么方法测？**

除了 ping 测试 IP，DMIT 官方 Looking Glass（`lg.dmit.sh`）可以选不同节点做路由追踪和测速文件下载。测速文件下载速度能直接反映实际带宽，晚高峰测一次白天测一次，对比两次结果。

**Q：Eyeball 线路稳定吗？路由会不会变？**

DMIT 官方说明，Pro 系列以外的高端优化线路有可能因网络攻击或成本原因切换路由。如果你对 CN2 GIA 有硬性要求，买 Pro 系列会更有保障；Eyeball 日常够用，但路由切换的风险要知道。

**Q：退款政策是什么？**

购买 3 天内且流量使用不超过 30GB，支持全额退款（会扣支付网关手续费）。超过 3 天、30 天内，按剩余价值比例退。
